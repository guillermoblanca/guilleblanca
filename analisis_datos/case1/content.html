En este caso práctico aprenderemos a leer datos desde un archivo Excel utilizando Pandas, una de las librerías más populares para análisis de datos en Python.

Paso 1: Importar las librerías

import pandas as pd

Paso 2: Cargar el archivo Excel

Suponiendo que el archivo se llama datos.xlsx y está en la misma carpeta del proyecto:

df = pd.read_excel('datos.xlsx')

Paso 3: Ver las primeras filas del DataFrame

print(df.head())

Paso 4: Analizar los datos

Podemos obtener información general del archivo, como columnas, tipo de datos y estadísticas descriptivas:

print(df.info())
print(df.describe())

Paso 5: Filtrar datos

Por ejemplo, si queremos filtrar todas las filas donde la columna Edad sea mayor a 30:

df_mayores_30 = df[df['Edad'] > 30]
print(df_mayores_30)

Conclusión

Con Pandas, leer y analizar archivos Excel es muy sencillo. Puedes aplicar filtros, agrupaciones y transformaciones de forma eficiente, lo cual facilita el análisis de datos.
