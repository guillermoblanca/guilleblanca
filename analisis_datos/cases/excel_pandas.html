Este ejemplo muestra cómo leer un archivo Excel usando pandas en Python: 


import pandas as pd

df = pd.read_excel('datos.xlsx')
print(df.head())

De esta manera debe mostrarse una tabla con los datos. head() permite visualizar los primeros valores de la tabla.
Para depurar se usa .info() o describe() devolviendo información adicional sobre la tabla así como los valores nulos que pueda contener.
